## 简介

受「中国制霸生成器」启发，制作的命运扳机大手子检定工具。

选择你对 10 位 Awakener 的熟练程度，算出你的阶位总分，生成可分享的结算图！

## 功能

- 点击角色色块选择熟练度（0~4）
- 本命仅可选一个
- 实时计算总分 + 显示评语
- 结算区展示本命角色像素 ICON
- 📸 一键生成并下载分享图片
- 自动保存状态（localStorage）
